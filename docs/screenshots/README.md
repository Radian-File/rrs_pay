# Screenshot placeholders

Add screenshots here only when they are intentionally generated from a polished build.

Recommended portfolio captures:

1. **Dashboard overview** — app shell, navigation, KPI cards.
2. **Employee management** — employee table and edit/create flow.
3. **Payroll workspace** — payroll period, totals, and actions.
4. **Reports/exports** — generated report or export confirmation.
5. **Settings and backup/restore** — saved settings plus backup controls.
6. **Audit logs** — filtered audit activity with export status.

Capture notes:

- Use demo data only; avoid real payroll or bank information.
- Keep the database path and personal Windows username out of cropped screenshots when possible.
- Prefer a consistent 16:9 viewport/window size for portfolio images.
- If screenshots are not meant to be committed, place them outside the repository or leave this folder as notes only.
